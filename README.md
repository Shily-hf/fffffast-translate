# fffffast Translate

一款轻量级的 Chrome 浏览器翻译扩展，专为英语学习者设计。鼠标悬停即可快速翻译单词，让你在阅读英文文章、文档时更加流畅，提升英语学习效率。

## 功能特点

- **悬停即译** - 鼠标悬停在英文单词上，自动显示中文翻译
- **多翻译源** - 支持 Google 翻译、DeepL、有道翻译、百度翻译
- **快捷开关** - 按 `ESC` 关闭，双击 `ESC` 重新开启
- **简洁配置** - 点击扩展图标，下拉选择即可切换翻译源

## 安装方法

1. 下载本项目到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹

## 使用方法

### 基本使用

安装后默认使用 Google 翻译，无需配置：

1. 打开任意英文网页
2. 将鼠标悬停在想要翻译的单词上
3. 翻译结果自动显示在单词上方

### 切换翻译源

1. 点击工具栏中的扩展图标
2. 选择翻译源
3. 如需 API 密钥，填写后点击「保存」
4. 点击「测试」验证配置

### 快捷键

| 按键 | 功能 |
|------|------|
| `ESC` | 关闭翻译 |
| `ESC` × 2 | 开启翻译 |

## 翻译源对比

| 翻译源 | 免费 | 需配置 | 说明 |
|--------|:----:|:------:|------|
| Google | ✓ | ✗ | 开箱即用 |
| DeepL | 额度 | ✓ | 翻译质量高 |
| 有道 | 额度 | ✓ | 国内访问快 |
| 百度 | 额度 | ✓ | 有 QPS 限制 |

## API 申请

- **DeepL**: https://www.deepl.com/pro-api
- **有道翻译**: https://ai.youdao.com/
- **百度翻译**: https://fanyi-api.baidu.com/
